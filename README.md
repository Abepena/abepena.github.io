# Abe Peña Portfolio

Personal portfolio site. Static HTML/CSS with React + Babel standalone for component rendering.

## Stack
- Plain `Portfolio.html` entry point
- React 18 + Babel Standalone loaded via unpkg
- CSS in `styles.css`, component sources in `src/*.jsx`
- Images in `assets/`

## Local preview
```
python3 -m http.server 8765
```
Open http://localhost:8765/Portfolio.html
